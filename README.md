# ShakaCode Trust-Based Commercial Licensing

ShakaCode products are built for professional teams, but we want developers to
learn, evaluate, demo, and adopt them without friction.

Our general rule:

> Free to learn, evaluate, demo, and use for qualifying open-source projects.
> Paid when the software creates private business value.

This repository explains ShakaCode's shared licensing philosophy and provides
reusable templates for ShakaCode products.

Product-specific license files control. This repository does not replace the
binding license terms in any product repository.
