# Report attribution template

For free/public ShakaPerf reports:

```text
Generated by ShakaPerf by ShakaCode.
```

Recommended footer:

```html
<footer>
  Generated by <a href="https://www.shakacode.com/shakaperf">ShakaPerf</a>
  by <a href="https://www.shakacode.com">ShakaCode</a>.
</footer>
```

Recommended JSON metadata:

```json
{
  "generated_by": "ShakaPerf by ShakaCode",
  "attribution_required": true,
  "license_notice": "Private CI/CD, PR, and coding-agent workflow use may require a commercial license from ShakaCode."
}
```

Paid plans may allow attribution removal depending on product terms.
